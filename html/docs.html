Quick Start

Get started with VNC-20 in minutes. Follow these simple steps to connect to the network and start building.

1. Install the VNC-20 SDK

npm install @vnc20/sdk ethers

2. Connect to the Network

import { VNC20Provider } from '@vnc20/sdk';

// Connect to mainnet
const provider = new VNC20Provider({
  network: 'mainnet',
  rpcUrl: 'https://rpc.vnc20.io'
});

// Get network info
const networkInfo = await provider.getNetworkInfo();
console.log(networkInfo);

3. Deploy Your First Contract

// Deploy a simple smart contract
const contract = await provider.deployContract({
  abi: contractABI,
  bytecode: contractBytecode,
  args: [initialValue]
});

console.log('Contract deployed at:', contract.address);

Consensus Mechanism

VNC-20 uses a hybrid Delegated Proof-of-Stake (DPoS) with Byzantine Fault Tolerance (BFT) consensus mechanism.

Key Features

  • Deterministic Finality: Transactions are final within 0.5-1 second
  • 87 Active Validators: Selected through token-weighted voting
  • BFT Safety: Can tolerate up to 28 Byzantine validators (‚Öì threshold)
  • Energy Efficient: No mining required, minimal energy consumption

Validator Selection

Validators are selected based on:

  1. Total VNC tokens staked
  2. Voting power from delegators
  3. Historical uptime and performance
  4. Minimum stake requirement: 100,000 VNC
‚ÑπÔ∏è Note: Validator elections occur every epoch (approximately 24 hours). The top 87 validators by voting power are selected for the next epoch.

Smart Contracts

VNC-20 is 100% EVM compatible. Deploy your existing Ethereum contracts without modification.

Example: VNC-20 Token

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract VNC20Token {
    string public name = "My Token";
    string public symbol = "MTK";
    uint8 public decimals = 18;
    uint256 public totalSupply;
    
    mapping(address => uint256) public balanceOf;
    mapping(address => mapping(address => uint256)) public allowance;
    
    event Transfer(address indexed from, address indexed to, uint256 value);
    event Approval(address indexed owner, address indexed spender, uint256 value);
    
    constructor(uint256 initialSupply) {
        totalSupply = initialSupply * 10 ** decimals;
        balanceOf[msg.sender] = totalSupply;
    }
    
    function transfer(address to, uint256 value) public returns (bool) {
        require(balanceOf[msg.sender] >= value, "Insufficient balance");
        balanceOf[msg.sender] -= value;
        balanceOf[to] += value;
        emit Transfer(msg.sender, to, value);
        return true;
    }
    
    function approve(address spender, uint256 value) public returns (bool) {
        allowance[msg.sender][spender] = value;
        emit Approval(msg.sender, spender, value);
        return true;
    }
}

Staking Guide

Stake your VNC tokens to help secure the network and earn rewards.

Staking Requirements

  • Minimum Stake: 100 VNC (for delegators)
  • Validator Stake: 100,000 VNC minimum
  • Lock Period: 30 days (flexible withdrawal after lock)
  • Current APY: 15-20% depending on validator performance

How to Stake

  1. Visit the VNC-20 Wallet
  2. Connect your wallet (MetaMask, WalletConnect, etc.)
  3. Select "Stake" and choose your preferred validator
  4. Enter the amount and confirm the transaction
  5. Start earning rewards immediately!
üí∞ Rewards Distribution: Staking rewards are distributed every epoch (24 hours) and automatically compound.

Setup Instructions

# Extract and install
tar -xzf vnc20-node.tar.gz

# Initialize the node
vnc20-node init --network mainnet

# Start the validator
